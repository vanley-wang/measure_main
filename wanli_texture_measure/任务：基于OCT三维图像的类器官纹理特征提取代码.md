# 任务：基于OCT三维图像的类器官纹理特征提取代码

## 1. 背景与目标
我正在做肿瘤类器官药物评估的研究。已有OCT三维体数据，需要为每个分割好的类器官提取三维纹理特征，用于后续药效建模。

## 2. 输入数据说明
- 数据格式：三维numpy数组，形状为 (Z, Y, X)，例如 (400, 800, 800)
- 数据类型：强度值，已归一化到0-255之间（uint8或float32）
- 每个类器官已有一个三维掩膜（相同形状的布尔数组或整数标签数组），掩膜区域即为该类器官的实体区域（不含空腔）
- 我已有：100个类器官的强度数据和掩膜，以列表或字典形式存储

## 3. 需要实现的功能

### 3.1 三维灰度共生矩阵（GLCM）特征
- 在三维空间内计算GLCM，考虑到计算效率，可以在三个正交方向（XY、XZ、YZ）分别计算2D GLCM，然后取平均值作为该体素/该区域的最终特征
- 使用步长=1，灰度级=32（将0-255压缩到32级）
- 需要提取的特征包括：
  - 对比度（Contrast）
  - 能量（Energy，也叫Angular Second Moment）
  - 熵（Entropy）
  - 同质性（Homogeneity）
  - 相关性（Correlation）
- 注意：只对掩膜内的体素计算，背景（掩膜外）不参与

### 3.2 三维灰度游程矩阵（GLRLM）特征【可选，如果不复杂就做】
- 同样在三个正交方向计算后取平均
- 提取特征：
  - 短游程强调（Short Run Emphasis, SRE）
  - 长游程强调（Long Run Emphasis, LRE）
  - 游程不均匀度（Run Length Non-Uniformity, RLN）
- 灰度级同样压缩到32级

### 3.3 小波多尺度特征（可选）
- 使用Daubechies小波（db4）对ROI区域进行2层分解
- 提取各子带（LL, LH, HL, HH）的能量和标准差

### 3.4 辅助功能
- 一个批量处理函数，输入：强度数组、掩膜数组列表 → 输出：特征DataFrame，每行是一个类器官，每列是一个特征
- 特征可视化：生成热力图展示特征值分布

## 4. 代码要求
- 语言：Python 3.8+
- 主要依赖库：numpy, scipy, scikit-image, PyWavelets（如果做小波）, pandas, matplotlib
- 代码风格：包含函数定义和注释，每个函数有docstring
- 输出：直接输出可运行的代码，并附带一个简单的测试示例（用随机生成的小三维数组模拟测试）
- 注意：如果某个特征计算量太大，请注明并给出替代方案

## 5. 输出格式
请按以下顺序输出：
1. 导入库部分
2. GLCM相关函数（含方向平均）
3. GLRLM相关函数（可选）
4. 小波特征函数（可选）
5. 批量处理主函数
6. 一个简单的验证测试代码
7. 使用说明（如何调用、参数调整建议）
